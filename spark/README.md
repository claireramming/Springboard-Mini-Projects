# Spark Mini-Project 
This repository contains all resources for the Spark mini-project.  
Skills used include:  
* reading data into spark context
* building lists
* lambda functions
* using .filter(), .map() and .flatmap()
* using .collect() and .take(#)
* using .reduce() and .reduceByKey()  
* visualizing data with seaborn
* dataframe and machine learning tutorial
